# Dinantia API

Dinantia API Documentation

**API version:** `1.2.0`

**Base URL:** `https://yourdomain.dinantia.com/api/web`

Within this page you'll find a list of available endpoints along with explanations and examples for Dinantia's [RESTful](https://en.wikipedia.org/wiki/Representational_state_transfer) web API.

Required headers:

```
Accept: application/vnd.api+json
Content-Type: application/vnd.api+json
```

Authentication: [Basic Authentication](https://en.wikipedia.org/wiki/Basic_access_authentication). You can find your school's User and Secret in the General Settings section of your school in the web app.

```
User: <User>
Password: <Secret>
```
